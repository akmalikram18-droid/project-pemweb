Deskripsi Project
Haven PetShop merupakan website online pet shop yang bertujuan untuk memudahkan pengguna dalam mencari informasi serta melakukan pemesanan kebutuhan hewan peliharaan secara online. Website ini dirancang dengan tampilan sederhana, responsif, dan user friendly menggunakan teknologi web dasar.

---

Fitur Website
- Navigasi Utama
- Halaman Home
- Filter/Kategori Produk
- Tombol “View Details”
- Halaman Tentang Kami
- Halaman Login
- Halaman Kontak

---

Teknologi yang Digunakan
- HTML
- CSS
- Bootstrap
- JavaScript
- Vercel (hosting)
